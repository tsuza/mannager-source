Why MANNager

Everything you need to
run a server.

One-click Setup
Download and configure any Source Engine server in minutes. DepotDownloader is handled for you automatically.
Port Forwarding
Automatic UPnP and SDR relay support so players around the world can connect without router headaches.
Live Console
Built-in server terminal, command history.
Sourcemod
Download Sourcemod stable or dev branch with a single click, directly integrated into the manager.
Live Editing
Edit server name, password, map, max players, port, and GSLT token inline. No config files needed.
Multi-Server
Run and monitor multiple server instances simultaneously with per-server status and controls.
Auto-Update
Keep your server files up to date with the built-in update panel: progress bar, live logs included.
Cross-Platform
Native support for Linux and Windows. One app, wherever you run your server.
How it works

Up and running in
three steps.

1
Configure
Give your server a name, pick a game (TF2, CS:GO, CS:S and more), and choose where to install it.
2
Download
MANNager runs DepotDownloader for you & streams the progress.
3
Launch
Hit play. Your server is live, and if selected, with automatic port forwarding ( or SDR ).
